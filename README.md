# Python Learning & Aerospace Practice 🚀

Welcome to my Python learning repository! This repository contains my journey of learning Python, ranging from basic programming concepts to more advanced topics like object-oriented programming (OOP) and aerospace-related coding exercises.

## 📌 Topics Covered
- **Basic Python Concepts**: Variables, Loops (`for`, `while`), Functions.
- **Data Structures**: Lists, Tuples, Dictionaries.
- **Object-Oriented Programming (OOP)**: Classes, Objects, Inheritance, Custom Exceptions.
- **File I/O**: Reading and writing various file formats (`txt`, `csv`, `xlsx`).
- **Aerospace Practice**: Simulating orbital mechanics, physics engines, flight computers, radar, and spacecraft (e.g., Voyager).

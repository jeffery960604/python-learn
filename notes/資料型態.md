# Python 資料型態深度解析

在 Python 中，萬物皆物件（Everything is an object）。理解資料型態不僅僅是知道它們的名稱，更重要的是理解它們在記憶體中的行為模式。

為了讓您能有邏輯地掌握，我們首先需要引入 Python 資料型態最核心的分類概念：「可變性（Mutability）」。

## 核心概念：可變 (Mutable) vs. 不可變 (Immutable)

在深入各別型態之前，這是您必須先建立的底層邏輯：

- **不可變型態 (Immutable)**：一旦在記憶體中被建立，其**內容就不能被改變**。若您試圖改變它，Python 實際上會在記憶體中創立一個「新的物件」，並將變數重新指向這個新物件。
    
- **可變型態 (Mutable)**：建立後，您可以直接**在原記憶體位址修改其內容**（新增、刪除、更改元素），而不會產生新的物件。
    

理解這一點，對於後續處理函數參數傳遞、複製（Copy）、以及作為字典鍵值（Dictionary Key）的規則至關重要。

## 系統化分類與舉例

我們依據資料的用途與結構，將 Python 內建的資料型態分為以下幾大類：

### 1. 數值型態 (Numeric Types) - `不可變`

用於表示數字，Python 支援極高精度的運算。

- **整數 `int`**：表示沒有小數點的數值。Python 的 `int` 支援任意大小（不受 32-bit/64-bit 限制，僅受限於記憶體大小）。
    
- **浮點數 `float`**：表示帶有小數點的數值（依循 IEEE 754 雙精度標準）。
    
- **複數 `complex`**：包含實部與虛部，常用於科學運算。
    

```
# 數值型態範例
age = 30                # int
temperature = 36.5      # float
math_complex = 3 + 4j   # complex

print(type(age))        # 輸出: <class 'int'>
```

### 2. 布林型態 (Boolean Type) - `不可變`

實際上是整數的子類別，用來表示真或假，主要用於邏輯判斷。

- **布林值 `bool`**：只有兩個值 `True`（底層視為 1）和 `False`（底層視為 0）。
    

```
# 布林型態範例
is_valid = True
is_empty = False

print(True + 1)         # 輸出: 2 (證明 True 底層行為類似數值 1)
```

### 3. 字串型態 (Text Sequence Type) - `不可變`

- **字串 `str`**：由 Unicode 字元組成的序列。因為是不可變的，任何字串操作（如拼接、替換）都會產生新字串。
    

```
# 字串型態範例
greeting = "Hello, Python!"
# greeting[0] = "h"  # 這會引發 TypeError，因為字串不可變

# f-string (格式化字串，Python 3.6+)
name = "Alice"
welcome_msg = f"Welcome, {name}!" # 輸出: Welcome, Alice!
```

### 4. 序列型態 (Sequence Types)

用來存放多個元素的有序集合，可透過索引（Index）來存取。

- **串列 `list` (`可變`)**：使用方括號 `[]`，可以容納不同型態的資料，且可以隨意增刪改。
    
- **元組 `tuple` (`不可變`)**：使用圓括號 `()`。一旦建立就無法修改，因此**讀取速度比 list 快**，且可作為字典的 Key。
    
- **範圍 `range` (`不可變`)**：通常用於迴圈，表示一個不可變的數字序列，它不會在記憶體中一次生成所有數字，而是「按需生成（惰性求值）」，非常節省記憶體。
    

```
# 序列型態範例
my_list = [1, "apple", 3.14]
my_list.append("new_item")      # 可以修改 (Mutable)

my_tuple = (1, "apple", 3.14)
# my_tuple[0] = 2               # 引發 TypeError (Immutable)

my_range = range(0, 10, 2)      # 產生 0, 2, 4, 6, 8
```

### 5. 映射型態 (Mapping Type)

- **字典 `dict` (`可變`)**：使用大括號 `{}` 存放鍵值對（Key-Value pairs）。底層實作是雜湊表（Hash Table），因此**查詢速度極快（O(1)）**。
    
    - _重要規則_：Dict 的 Key 必須是「不可變型態」（如字串、數字、元組），因為需要計算 Hash 值；Value 則可以是任意型態。
        

```
# 字典型態範例
user_info = {
    "name": "Bob",
    "age": 25,
    "skills": ["Python", "Data Analysis"] # Value 可以是 List
}

print(user_info["name"])        # 輸出: Bob
user_info["age"] = 26           # 可以修改 (Mutable)
```

### 6. 集合型態 (Set Types)

用於存放「不重複」元素的無序集合，底層同樣基於雜湊表實作。常應用於去重（Deduplication）或數學集合運算（交集、聯集）。

- **集合 `set` (`可變`)**：使用大括號 `{}`（但沒有鍵值對）或 `set()`。
    
- **凍結集合 `frozenset` (`不可變`)**：不可變版本的集合，因此它可以作為字典的 Key。
    

```
# 集合型態範例
my_set = {1, 2, 2, 3, 3}
print(my_set)                   # 輸出: {1, 2, 3} (自動去重)

my_set.add(4)                   # 可以修改 (Mutable)

# 集合運算
set_a = {1, 2, 3}
set_b = {3, 4, 5}
print(set_a & set_b)            # 輸出: {3} (交集 Intersection)
```

### 7. 空值型態 (None Type) - `不可變`

- **空值 `NoneType`**：只有一個值 `None`。用來表示「沒有值」、「空」或「函數無回傳值」的狀態。這與空字串 `""` 或數字 `0` 在邏輯上是完全不同的。
    

```
# 空值型態範例
result = None

def do_nothing():
    pass

print(do_nothing())             # 輸出: None
```
